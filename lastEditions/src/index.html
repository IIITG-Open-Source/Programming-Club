About The Contest

Welcome to a battle of designs. CSS Clash is a competition organized by the Programming club of IIITG where participants modify this page using their CSS Skills to provide the best visual look. To get started, download the HTML code of this page to work locally on your system. Give this page your desired styling by writing CSS in a new file with name "style.css". Note: You cannot change the HTML of this page. Once finished, submit your CSS file through the google form linked above. The Top 3 Entries will win cash prizes of Rs.400 each!

Rules

  • Only 1 submission is allowed per participant
  • Deadline to submit your design is Sunday, 14th November, till 11:59 PM
  • You are free to inspire your design from any resources
  • You can only submit the CSS code you write. No changes to the HTML will be submitted
  • You can use any tools/libraries/resources you want
  • You can insert background images or change any font styles etc as per your requirements
  • The Submissions will be judged on the basis of layout, design, effects and responsiveness
